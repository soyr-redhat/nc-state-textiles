## Marketing & Copywriting

### 1. Product Description Writer

```

Write a compelling product description for the following product:

Product Name: [name]
Key Features: [list 3-5 features]
Target Audience: [demographic, age, interests]
```

```
3. Then, [third step: evaluate/prioritize/hypothesize]
```
